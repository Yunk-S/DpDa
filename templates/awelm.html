Adaptive Weighted Ensemble Learning Model (AWELM)

Build high-accuracy disease prediction models through gradient descent optimization of ensemble weights

Loss Function and Weight Optimization
Loss Function: L(w) = -mean[β·y·log(p) + (1-y)·log(1-p)] + λ·H(w)
  where p = Σwᵢ·pᵢ (weighted probability), β = neg/pos ratio (class balancing factor)
  H(w) = -Σwᵢ·log(wᵢ) (negative entropy regularization, prevents weight overfitting)

Optimization Objective: min L(w)  s.t. wᵢ ≥ 0, Σwᵢ = 1

Base Models: Logistic Regression + Random Forest + Gradient Boosting + SVM
